# 🎬 Movie Data Analysis Project (Python)

This project performs *Exploratory Data Analysis (EDA)* on the *TMDB 5000 Movies Dataset* using Python.  
The aim of this project is to analyze movie budgets, revenues, ratings and genre distribution using *data analysis and visualization techniques*.

This project is built using *Pandas, Matplotlib and Seaborn* to understand patterns and insights in the movie industry.

---

## 📊 Project Objectives

The main objectives of this project are:

- Load and explore the movie dataset
- Clean the dataset by handling missing values
- Analyze the relationship between *movie budget and revenue*
- Identify *top performing movies*
- Analyze *average ratings across genres*
- Visualize *genre distribution*

---

## 📁 Dataset

Dataset used: *TMDB 5000 Movies Dataset*

It contains information such as:

- Movie Title
- Budget
- Revenue
- Genres
- Vote Average (Ratings)
- Release information

## 📈 Analysis & Visualizations

The following analyses are performed in this project:

### 1️⃣ Distribution of Movie Budgets
A histogram is used to visualize how movie budgets are distributed.

### 2️⃣ Budget vs Revenue Analysis
A scatter plot is used to examine whether higher budgets lead to higher revenue.

### 3️⃣ Average Rating by Genre
Genres are split and analyzed to calculate the *average rating per genre*.

### 4️⃣ Top 10 Movies by Revenue
A bar chart identifies the *highest grossing movies*.

### 5️⃣ Genre Distribution
Pie charts and donut charts show the *distribution of movie genres* in the dataset.
